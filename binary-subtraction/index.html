Binary Subtraction

Like addition, subtractions also plays an important role in binary arithmetic as well as in digital electronics system. We have discussed about the binary subtraction process in brief in the article binary arithmetic. But in this article we will look into the process of binary subtraction only and more elaborately.
Coming directly to the topic without wasting any time, first we have to look into the four fundamental steps of binary subtraction. These are

0 – 0 = 0
0 – 1 = 1, borrow 1 from the next more significant bit
1 – 0 = 1
1 – 1 = 0

As there are only two digits in binary number system 0&1 that’s why these four steps are able to describe all the operations of binary subtraction.
