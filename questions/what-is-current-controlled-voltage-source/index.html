What is current controlled voltage source?

An energy source is must for every active electrical circuit. Electrical energy source is classified into two types. One is the independent source and another is the dependent source. Independent sources include voltage and current source, where the output voltage or current is totally independent from the behavior of connected load. We can also define ideal voltage and ideal current sources from here. Dependent sources are also called as controlled sources, where the voltage or current is depended upon the load. In this type the source, voltage or current is not fixed but is dependent on a voltage or current at some other location in the circuit. Dependent sources are of four types:

Voltage controlled voltage source (VCVS)
Current controlled voltage source (CCVS)
Voltage controlled current source (VCCS)
Current controlled current source (CCCS)

In this article we will discuss about the current controlled voltage source which is also written as CCVS. As the name suggests that it is a voltage source which is controlled by current. This kind of source is unilateral. In current controlled voltage source the output voltage is controlled by the input current. Electronic systems that uses either the transistor (FET) or vacuum tubes, needs dependent sources like CCVS. Load current signal is fed to the input and by semiconductor control, the output voltage is determined. A current controlled voltage source has the unit of conductance that is Siemens. It is also called as transconductance. If both the independent and dependent variable are same (may be either voltage or current) then the proportionality constant is dimensionless.
