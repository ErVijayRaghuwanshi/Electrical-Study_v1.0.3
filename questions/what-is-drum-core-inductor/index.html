What is drum core inductor?

Drum core is used to describe the shape of the ferrite core used in some power inductors, and it differentiates these inductors from other types such as toroidal, power bead, flat coil, or planar. Actually the shape of the core is different for various types of inductor. Various shapes of inductors are used for various types of applications. There are two types of drum core inductors, unshielded type and shielded type. Among them shielded type drum core inductors are used widely as this type of inductor provides better safety. These devices are typically used in applications such as catalog power modules, computer equipment, telecom equipment, and consumer electronics such as game station consoles, digital cameras, PDAs, CD players and LCD displays and TVs.
