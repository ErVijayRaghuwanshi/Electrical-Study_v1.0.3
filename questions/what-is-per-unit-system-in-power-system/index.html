What is per unit system in power system?

Per unit system use in power system to simplify hard calculation.It is the method of expressing quantities in an electrical system as a proportion of predefined base quantities. The per-unit value of a quantity is the ratio of the original quantity to its base value.

where Qpu is per unit quantity,

Q is quantity in normal unit,

Qbase is the base value of quantity in normal unit.

Let the base value of current is 200 amp then the per unit value of 100 amp current is 100 / 200=0.5.
