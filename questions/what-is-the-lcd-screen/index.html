What is the LCD screen

Actually, full form of LCD is Liquid Crystal Display. LCD screen is an electronic display module used in a wide range of sophisticated electronics appliances. LCD modules are more popular than seven segment display and any other multi segment LEDs because:

  1. LCDs are very cheap.
  2. Thay are easily programmable.
  3. They are able to display special characters & animations.

A 16 × 2 LCD display is very basic module and this module is widely used in several electronic devices.

Actually, a 16 × 2 LCD means it is able to display 2 lines & 16 characters can accommodate per line.

An typical LCD has two registers, namely, Command register and Data register.

The ' command register' stores the commands provided to the LCD. Eg of command instructions are initializing, setting the cursor position etc. Whereas, the data register stores the data bytes to be displayed on the LCD.
