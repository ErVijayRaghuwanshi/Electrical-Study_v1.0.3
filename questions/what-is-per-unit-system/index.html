What is per unit system ?

Any calculation is made easy if we consider base parameters. Similarly in case of a power system there are different voltage and power levels which are connected with each other via transformers (both step-up and step-down). Now if we take the absolute quantities for analysis purpose, the calculation becomes very tiring. So to make the work easy, the concept of per unit system has been introduced. To find out the p.u. value a base or reference value in the same unit is assumed. Now if we calculate the ratio of the actual value of the quantity and the base value as unit, we will get per unit value and when we represent the whole network by the per unit values, that system is termed as per unit system.
