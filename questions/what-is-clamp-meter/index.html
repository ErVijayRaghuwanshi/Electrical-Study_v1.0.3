What is clamp meter?

Clamp meters are the testing instruments which is used for measuring current. Current on a live conductor can be measured using a clamp meter. The speciality of clamp meter is that it can measure the current value without any interruption. Clamp meter is thus also named as current clamps. It can measure large amounts of current. Both ac and dc current measurement can be obtained using this clamp meter. This device can simply measure current by clamping on a conductor. Thus the name clamp is justified. Clamp meters also measures voltages both ac and dc. Resistance, capacitance, frequency can also be measured using the clamp meter. In industry electricians, technicians or contractors use clamp meter for different works in their field. These are installation, service and maintenance.

Current clamp can be 3 different types. They are:

  1. Current transformer (C.T)
  2. Iron vane type
  3. Hall effect type

Clamp meter can measure easily currents from the range of hundreds of amperes to several mili amperes. Now a days digital clamp meter is available in which the reading is obtained in digital format. It is more convenient to use than an ordinary multimeter. Sometimes multimeter are provided with inbuilt current clamp.
