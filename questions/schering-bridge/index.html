Schering bridge

Measurement of inductance and capacitance can be made conveniently and accurately by employing a.c bridge networks . The ac bridges are modified form of Wheatstone's bridge which contains four arms (impedances), a source voltage and a balance detector. Generally the capacitance of any network can be measured by using Desauty's bridge and Schering bridge. Now we will discuss the Schering's bridge in brief.

Here ,

  • Z1 = R1 / 1 + j ω C1 R1
  • Z2 = R2
  • Z3 = - j / ω C3
  • Z4 = R4 / 1 + j ω C4 R4

At balance condition , Z1 Z4 = Z2 Z3

Putting the values we get , R1 / 1 + ω2 C12 R12 = C4 R2 / C3 ...............(i)

From vector diagram we can see that C1 and R1 are in parallel,

cos2δ = ω2 C12 R12 / 1 + ω2 C12 R12

Substituting the value of cos2δ in the equation (i) we have,

cos2δ / ω2 C12 R1 = C4 R2 / C3

C1 = C3 cos2δ / ω2 C1 C4 R1 R2 ........ (ii)

By the vector diagram we can get the value of R4 and substituting the value of R1 in (ii) we get C1 where δ is the loss angle of the capacitor.
