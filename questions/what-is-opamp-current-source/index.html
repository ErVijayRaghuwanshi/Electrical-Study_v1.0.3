What is opamp current source?

The full form of opamp is operational amplifier. Sometime constant current operation is important for some special circuits. Transistor based current sources are the partial solution of this, but something more sophisticated is required in order to increase the stability of high gain circuits. An opamp current source is actually a modified version of conventional transistor operated current source. The base emitter junction of the circuit trasistor is fed to the Opamp, in this way it becomes a non inverted amplifier circuit with constant input voltage. In more easy way, there are two inputs of an Opamp, called as inverting and non inverting input. But the output is one. The collector and amitter goes to the input terminals of the Opamp. And the output of Opamp is taken and fed to the base of the transistor.

Opamp is actually acts like an amplifier circuit. AS the opamp output is connected with the base emitter junction, so the base emitter voltage increases automatically. But, limiting the current is very essential as this circuit has application in constant current operations. So, a constant current sensor resistor is connected with the citcuit. In this way, the Opamp automatically adjusts the constant current with the help of sensing resistor and zener diode.
