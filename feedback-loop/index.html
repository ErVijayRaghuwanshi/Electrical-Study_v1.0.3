Feedback Loop

A control system is a collection of different elements which is designed to perform a specific task that is on giving a particular input it gives an output in its response. The input is a reference input and the output is a controlled output. This cause (input) and effect (output) relationship is governed by a mathematical relation. This relation may be linear or non linear in nature. A control system is of two types – open loop control system and closed loop control system. In an open loop control system no part of the output is fed back to input. There is no feedback path. But in case of closed loop system a part of the output is returned back to input via a feedback path or feedback loop.

There are different elements which constitute the closed loop system. These systems are discussed below in brief. Feedback loop is a very important part in this closed loop control system.

• Reference input: the reference input r(t) is fed to the comparator.
• Comparator: continuously compares the reference input with the feedback which produces the error signal e(t)
• Controller: e(t) that is error signal is fed to the controller where is the signal undergoes manipulation. Outcome is manipulated signal m(t).
• Process or plant : m(t) is given to the process or plant where it is undergoes final processing to get desired controlled output c(t)
• Feedback part: a part of this controlled output is fed back to the comparator via feedback system . Feedback gain is obtained from the feedback system.

Feedback paths are of two different types –
• Positive (+ve) feedback loop - Positive feedback path increases gain of the amplifier. It makes the system to tend away from equilibrium and become unstable. In this case feedback signal and input signal are in same phase. In positive feedback the input part is reinforced to the output point. In this point the device oscillates between its maximum and minimum positions. The system with positive feedback loop can sense large signals while ignoring small ones. This may be due to introduction of hysteresis in the circuit. This type of feedback eliminates noise from digital signals. A device can be made to latch under certain conditions by positive feedback loop. There are many applications of this feedback path. Some of them are –
a) Use in electronics (in transistors, piezoelectric crystal, oscillators – Wien’s bridge , Hartley, Colpits oscillator, etc )
b) Use in electrical switches
c) In biology – physiology, gene regulation, evolutionary biology, immune system and psychology.
d) Human population rise, climatology, meterology, in chemistry

• Negative (-ve) feedback loop – Gain of the amplifier is reduced in case of negative feedback path. It makes the system more stable by keeping it to almost an equilibrium state. Here the feedback signal and input signal is not in phase, they are out of phase. Feedback signal is from 90 to 270 degree from the input signal. Negative feedback loop is used to correct errors that is minimizes the output errors. Waveform distortion can be corrected using this kind of feedback loop. Negative feedback path symbolizes asymptotic gain models. Some of the applications are as-
a) Used in electronic amplifiers (negative feedback amplifiers)
b) Economics
c) Chemistry and biology
d) Mechanical engineering (centrifugal governor)
e) Control system (thermostat control )
