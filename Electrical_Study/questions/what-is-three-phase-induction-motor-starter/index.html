What is three phase induction motor starter?

The word starter means that it is a device which is used for starting or beginning something. We know that a three phase induction motor is a self starting motor. When stator is supplied with 3-ph supply, a rotating magnetic field is produced. Then the rotor starts rotating. At the time of starting slip is unity so starting current is huge. The word starter is used as a misnomer here. The function of starter is not to start something here but to limit the value of huge starting current. Apart from choking huge starting current starter also helps in providing overload and under voltage protection. There are three common methods used for starting of squirrel cage induction motor. These are:

• Direct on-line starter (D.O.L): This is a simple and cheap method. In this method motor is connected by means of a starter across the full supply voltage. This starter also provides under voltage and overload protection. Final equation of DOL starter is that,τst / τfl = ( Ist / Ifl)2 sfl

• Star-delta starter: This is a common type of starter and is extensively used. Cage motor with delta connected stator winding uses this kind of starter. It uses two switches RUN and START. Here, the starting torque with star delta starting / full load torque with stator winding in delta is equal to 1/3( Ist / Ifl)2sfl
