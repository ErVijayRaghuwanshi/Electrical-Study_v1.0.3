How to find out RMS voltage?

There are different types of periodic waveforms. These can be represented by the equation f (t) = f (t+nT) where n is an integer and T is the time period. Out of these periodic functions, the sinusoidal function being the most common and being used extensively. Sinusoidal periodic function is an alternating quantity which is characterized by certain values or parameters like instantaneous value, peak value, phase, RMS value, form factor. RMS value is the topic of discussion here. RMS stands for Root Mean Square. The effective value of a sine wave is zero for the entire cycle. So in order to get the effective value, we compute the capability of the sine wave in terms of its heating power. In any alternation wave, frequency f is given as f = ω /2π where ω is the angular displacement in per unit time. The time period T is written as T = 1/f (f is in Hz, T in seconds). RMS voltage for sine wave (alternating) can be found out using the maximum or peak value.
RMS voltage is given as - Vrms = Vmax / √ 2. Thus RMS voltage can be also written as 0.707 times Vmax value. Vmax value is the max value of the wave during either positive or negative half cycle.
