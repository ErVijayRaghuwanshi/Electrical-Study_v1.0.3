Closed Loop Control System

n a closed loop control system the output has an effect on control action through the feedback system . Hence closed loop control system are also termed as feedback control system . The control action is actuated by an error signal e which is the difference between the input signal r and output signal c . This process of comparison between the output and input maintains the output at a desired level through control action process.The control system without involving human intervention for normal operation are called as automatic control systems. A closed-loop (feedback) control system using a power amplifying device prior to controller and the output of such a system being mechanical i.e position , velocity, acceleration is called servomechanism. There are different elements which constitute the closed loop system. These systems are discussed below in brief .

Reference input : the reference input r (t) is fed to the comparator .

Comparator : continuously compares the reference input with the feedback which produces the error signal e(t)

Controller : e(t) i.e error signal is fed to the controller where is the signal undergoes manipulation . Outcome is manipulated signal m(t).

Process / plant : m(t) is given to the process or plant where it is undergoes final processing to get desired controlled output c(t)

Feedback : a part of this controlled output is fed back to the comparator via feedback system . Feedback gain is obtained from the feedback system.

Transfer function of closed loop control system Y(s) / X(s) can be obtained in the following way

Z(s) = X(s) - Y(s) H(s)

Y(s) = Z(s) G(s)

Y(s) = G(s) X(s) - G(s) Y(s)H(s)

Y(s) [1+G(s)H(s) ] = G(s) X(s)

TRANSFER FUNCTION (T.F) = Y(s) / X(s) = G(s) / 1+G(s)H(s) …………… (i)

Equation (i) is the closed loop transfer function for negative ( -ve) feddback . In case of +ve feedback we have Transfer function T.F = Y(s) / X(s) = G(s) / 1-G(s)H(s) ………….. (ii)

Some examples of closed loop control system are : air conditioner , refrigerator , remote control of t.v ,air circulating system of an airplane, human body (practical example).

Closed loop control system has advantages as well as some drawbacks too . Advantages are as follows:

Accuracy is very high
Effect on non- linearity and distortion is reduced
Less sensitive to disturbances and environmental changes
Bandwidth ie operating frequency zone is very high

Disadvantages :

Complicated and time consuming from in design and hence costlier
It can become unstable under certain conditions
Due to feedback system tries to correct the error from time to time . Tendency to overcorrect the errror may cause oscillations .
